# InstantCrafting

A BepInEx 5.4.x (x64) mod for Moonlight Peaks to make all crafting, cooking, brewing, and production processes instantaneous.
This mod removes the waiting time, allowing you to craft, cook, brew, and produce items instantly.

## Features
- **Instant Crafting:** All items are crafted immediately without delay.
- **Instant Cooking:** All food is cooked immediately.
- **Instant Brewing:** All potions/drinks are brewed immediately.
- **Instant Production:** All production machines (Refinery, Press, Seed Sequencer, Chocolatier, etc.) process items immediately.
- **Configurable:** Fine-tune the behavior in-game using the BepInEx Configuration Manager.

## Requirements
- [BepInEx 5.4.x (x64)](https://github.com/BepInEx/BepInEx/releases)
- [Optional: BepInEx Configuration Manager](https://github.com/BepInEx/BepInEx.ConfigurationManager/releases)

## Installation
1. Ensure [BepInEx 5.4.x](https://github.com/BepInEx/BepInEx/releases) is installed in your Moonlight Peaks game folder.
2. Download the mod.
3. Extract the contents into your `MoonlightPeaks/BepInEx/plugins/` folder.

## Configuration
Upon first run, the mod will generate a configuration file located at `MoonlightPeaks/BepInEx/config/`. 

**Available Settings:**
- `Instant Crafting`: Enable or disable instant crafting. Default: True.
- `Instant Cooking`: Enable or disable instant cooking. Default: True.
- `Instant Brewing`: Enable or disable instant brewing. Default: True.
- `Instant Production`: Enable or disable instant production (Refinery, Press, Seed Sequencer, Chocolatier, etc.). Default: True.

*Optional: Use [BepInEx Configuration Manager](https://github.com/BepInEx/BepInEx.ConfigurationManager/releases) to edit settings conveniently in-game (default hotkey: F1).*
